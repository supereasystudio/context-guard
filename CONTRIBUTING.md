# Contributing to Context Guard 🛡️

We love contributions! Since this is an AI-delivered project, we have a few specific guidelines to keep the engine clean.

## How to Contribute

1.  **Report Bugs:** Open an issue if the token counting seems off for a specific file type.
2.  **Suggest Features:** Want support for a new LLM's pricing? Open an issue.
3.  **Pull Requests:** 
    - Keep it minimalist. 
    - Use `tiktoken` for all core logic.
    - Ensure any new UI elements match the **Teal/Black Magic UI** aesthetic.

## Development

The core logic is in `context_guard.py`. 

```bash
# Install dev dependencies
pip install tiktoken

# Run locally
python3 context_guard.py .
```
